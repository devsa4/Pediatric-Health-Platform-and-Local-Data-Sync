# Pediatric Health Platform & Local Data Sync : "Growth Guardian"

•	Architected a local-first web application using IndexedDB and MongoDB, guaranteeing database availability and complete app functionality during offline states.

•	Developed interactive pediatric telemetry visualization charts using Recharts, mapping height, weight against WHO Child Growth Standards.

•	Engineered a full-featured Progressive Web App (PWA) utilizing Service Workers to enable a complete offline uptime, reducing latency by 40%.
